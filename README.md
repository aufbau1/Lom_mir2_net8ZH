# Lom_mir2_net8ZH
水晶传奇三端版，C#脚本驱动带热更，带移动端，使用微端更新。
</br>
</br>
升级记录：
1、PC客户端图像渲染升级VorticeDX11；
2、废弃自管理二进制数据库，迁移Sqlite + MySQL切换；
3、废弃txt脚本，迁移C#脚本，预留所有业务逻辑脚本化能力；
4、增加C#自动热更（FileWatcher） + 手动推送；
5、增加C#脚本调试，增加变量单步跟踪；
6、增加AI辅助生成C#脚本；
7、客户端增加平滑移动；
8、客户端增加视角缩放；
9、增加安卓+iOS移动端（Xamarin + monogame），实现三端互通；
10、移动端自绘NPC对话、商店、任务、商城、任务、好友、邮件、组队、大地图；
11、增加移动端微端 + 移动端版本管理；
12、移动端废弃自绘HUD，改为FairyGUI；
13、增加PC端微端，PC端修改版本管理逻辑（与移动端保持一致）；
14、移动端升级MAUI，使用AOT编译；
15、移动端瘦身初始资源包，只保留初屏资源和系统自带库，apk体积1.3G降到85M；
